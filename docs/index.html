Quick Start

Up and running in under 2 minutes

# Clone the repository
git clone https://github.com/hardik139337/linkforge-starter-kit.git
cd linkforge-starter-kit

# Install dependencies
npm install

# Configure environment
cp .env.example .env

# Run it
npm run dev
