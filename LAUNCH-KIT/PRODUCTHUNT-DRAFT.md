## Description (paste this)

LinkForge Starter Kit is a production-ready SaaS boilerplate built on Next.js 16 that gives you everything you need to launch a SaaS product — auth, payments, database, analytics, landing page, and dashboard — all wired up and working out of the box.

Unlike skeleton templates with TODO comments, LinkForge ships as a complete, working link-in-bio SaaS application. Every integration is wired end-to-end: sign up, subscribe via Stripe, create links, view analytics. It works from the first `npm run dev`.

Built with the latest stack: Next.js 16 (App Router + Turbopack), TypeScript, Tailwind CSS v4, Prisma 7, NextAuth v5, and Stripe.

Use it as:
- A ready-to-launch link-in-bio SaaS (deploy and start selling)
- A battle-tested foundation for any SaaS project
- A reference architecture for production Next.js apps

Tech stack: Next.js 16 · TypeScript · Tailwind v4 · Prisma 7 · NextAuth v5 · Stripe · SQLite/PostgreSQL

## First Comment (maker comment — post immediately after launch)

Hey Product Hunt! 👋

I built LinkForge Starter Kit because I was tired of spending the first 2 weeks of every SaaS project wiring up the same infrastructure — auth, payments, database, landing page, dashboard.

Most boilerplates give you a skeleton with TODO comments. I wanted something different: a **complete, working SaaS application** where every integration actually works. Sign up works. Stripe checkout works. Webhooks work. Analytics work. You deploy it and start selling.

Here's what's included:
- NextAuth v5 with JWT sessions and protected routes
- Stripe checkout + webhooks + subscription management
- Prisma 7 with SQLite (dev) and PostgreSQL (prod)
- Full dashboard with CRUD, analytics, and live preview
- 5 profile themes (Minimal, Bold, Gradient, Neon, Glass)
- Conversion-optimized landing page with pricing
- 16 API routes, all typed and tested

Clone → install → `npm run dev` → you have a SaaS.
